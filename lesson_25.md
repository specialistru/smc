## 📘 Глава 25. Change of Character (CHoCH) – смена тренда

---

### 1. Заголовок, номер и краткое определение

**Название:** Change of Character (CHoCH) – смена тренда  
**Номер:** Глава 25 (Часть 5. Структура рынка, инструмент №3 из 7)  
**Суть одной фразой:** Change of Character (CHoCH) – это смена структуры рынка, которая происходит, когда цена пробивает последний значимый свинг в противоположном направлении и закрывается за ним, после чего формирует новый экстремум, подтверждая разворот тренда с восходящего на нисходящий или наоборот.  
**Визуальное представление:** 🔄📉➡📈 или 🔄📈➡📉 – оранжевая стрелка на баре с подтверждением CHoCH, часто сопровождающаяся пунктирной линией, разделяющей старую и новую структуру.

---

### 2. Исторический контекст и происхождение

Термин **Change of Character (CHoCH)** был введён и популяризирован **Майклом Хаддлстоном (ICT)** в его методологии как ключевое понятие для определения разворота тренда. В классическом техническом анализе аналогами являются «смена тренда», «прорыв линии шеи» (в паттернах голова-плечи) или «разворотная дивергенция». Однако ICT дал чёткое алгоритмическое определение: CHoCH происходит только после того, как цена сначала **собирает ликвидность** (Sweep) за последним свингом, а затем закрывается **за противоположным свингом**, меняя структуру.

В SMC Pro CHoCH является одним из самых важных инструментов, так как он отделяет ложные пробои (Sweep) от истинных разворотов. Без CHoCH любое движение может быть просто коррекцией.

**Почему CHoCH важен?**  
Розничные трейдеры часто воспринимают любое движение против тренда как разворот и входят слишком рано. CHoCH даёт **объективный сигнал**, что структура действительно сломана, и крупные игроки сменили направление. Понимание CHoCH позволяет:
1. Отличить разворот от простой коррекции.
2. Входить в сделку на ранней стадии нового тренда.
3. Избегать ложных сигналов на пробоях свингов.

---

### 3. Математическое определение и формальные условия

#### Определение CHoCH
Пусть:
- `lastSwingHigh` – последний значимый максимум перед предполагаемым разворотом.
- `lastSwingLow` – последний значимый минимум.

**Бычий CHoCH (разворот вверх):**
1. Цена сначала пробивает последний минимум (`low[i] < lastSwingLow`) – это сбор ликвидности (Sweep).
2. Затем цена закрывается **выше** последнего максимума: `close[i] > lastSwingHigh`.
3. После этого формируется новый минимум, который **выше** предыдущего минимума (подтверждение смены структуры).

**Медвежий CHoCH (разворот вниз):**
1. Цена пробивает последний максимум (`high[i] > lastSwingHigh`).
2. Закрывается **ниже** последнего минимума: `close[i] < lastSwingLow`.
3. Формируется новый максимум, который **ниже** предыдущего максимума.

#### Упрощённое определение (используется в некоторых реализациях)
CHoCH = пробой последнего свинга + закрытие за противоположным свингом.  
Третье условие (новый экстремум) часто проверяется неявно.

#### Отличие CHoCH от BOS:
| Характеристика | BOS | CHoCH |
|----------------|-----|-------|
| Требуется Sweep | Нет | Да (пробой противоположного свинга) |
| Смена тренда | Нет | Да |
| Возврат за противоположный свинг | Нет | Да |
| Подтверждение новым экстремумом | Нет | Да (неявно) |

#### Пример с реальными числами (бычий CHoCH, EURUSD H1):
- Последний Swing Low = 1.1000, последний Swing High = 1.1020.
- Бар 1: low=0.9995 (Sweep SSL), close=1.1015 (ещё не выше Swing High).
- Бар 2: close=1.1025 (выше Swing High) → CHoCH вверх.
- Бар 3: low=1.1010 (выше предыдущего минимума) – подтверждение.

### 4. Алгоритм обнаружения (псевдокод + реальный код из индикатора)

**Псевдокод:**
```
Функция DetectCHoCH(i, high, low, close, lastSwingHigh, lastSwingLow):
    // Бычий CHoCH
    если low[i] < lastSwingLow и close[i] > lastSwingHigh:
        вернуть 1 (бычий)
    // Медвежий CHoCH
    если high[i] > lastSwingHigh и close[i] < lastSwingLow:
        вернуть -1 (медвежий)
    иначе:
        вернуть 0
```

**Реальный код из индикатора SMC Pro (функция `DetectCHoCH`):**

```cpp
bool DetectCHoCH(int i, const double &high[], const double &low[], const double &close[], int lookback)
{
   // Находим последние свинги в пределах lookback
   double lastHigh = -1, lastLow = -1;
   for(int j = i-1; j >= 0; j--)
   {
      if(IsSwingHigh(j, high, lookback)) { lastHigh = high[j]; break; }
   }
   for(int j = i-1; j >= 0; j--)
   {
      if(IsSwingLow(j, low, lookback)) { lastLow = low[j]; break; }
   }
   if(lastLow > 0 && low[i] < lastLow && close[i] > lastHigh && lastHigh > 0)
      return true;  // бычий CHoCH
   if(lastHigh > 0 && high[i] > lastHigh && close[i] < lastLow && lastLow > 0)
      return true;  // медвежий CHoCH
   return false;
}
```

**Отрисовка CHoCH в SMC Pro (функция `DrawCHoCH`):**

```cpp
void DrawCHoCH(const datetime &time[], const double &open[], const double &high[],
               const double &low[], const double &close[], int totalBars)
{
   if(!InpShowCHoCH) return;
   for(int i = ObjectsTotal(0,0,-1)-1; i >= 0; i--)
   {
      string name = ObjectName(0,i);
      if(StringFind(name, g_prefix + "CHoCH_") == 0)
         ObjectDelete(0, name);
   }
   for(int i = totalBars-3; i >= InpSwingLookback; i--)
   {
      if(DetectCHoCH(i, high, low, close, InpSwingLookback))
      {
         bool bullish = (close[i] > open[i]);
         double price = bullish ? low[i] : high[i];
         DrawArrow(g_prefix + "CHoCH_" + IntegerToString(time[i]), time[i], price, 
                   bullish ? 1 : -1, MS_COLOR_CHOCH, 2);
      }
   }
}
```

**Пояснения:**  
- `MS_COLOR_CHOCH` по умолчанию – оранжевый (`clrOrange`).
- Стрелка рисуется на баре, на котором выполнены условия CHoCH.
- В некоторых версиях дополнительно рисуется прямоугольник, охватывающий зону разворота.

### 5. Как это выглядит на графике

#### Сценарий 1. Бычий CHoCH (разворот вверх)

**График EURUSD H1:**
- Нисходящий тренд: последний Swing Low=1.1000, Swing High=1.1020.
- Свеча A: low=0.9995 (сбор ликвидности SSL), close=1.1015 (ещё не выше Swing High).
- Свеча B: close=1.1025 (выше Swing High) → индикатор рисует **оранжевую стрелку вверх** на свече B.
- Свеча C: low=1.1015 (выше предыдущего минимума) – подтверждение.
- Затем рост до 1.1050.

#### Сценарий 2. Медвежий CHoCH (разворот вниз)

- Восходящий тренд: Swing High=1.1050, Swing Low=1.1020.
- Свеча A: high=1.1055 (Sweep BSL), close=1.1030.
- Свеча B: close=1.1015 (ниже Swing Low) → оранжевая стрелка вниз.
- Падение до 1.0980.

#### Как отображается в SMC Pro:
- Оранжевая стрелка (вверх или вниз) на баре, где произошло закрытие за противоположным свингом.
- Часто сопровождается штриховой линией, отделяющей старую структуру от новой.

### 6. Торговая тактика (полный цикл сделки)

CHoCH – это **надёжный сигнал разворота**. Стратегия: вход на закрытии свечи CHoCH (или на следующей свече) в направлении нового тренда.

#### Стратегия «CHoCH разворот»

**Шаг 1. Идентифицируем CHoCH**
- На графике видим оранжевую стрелку. Убеждаемся, что перед этим был sweep противоположного свинга.

**Шаг 2. Определяем контекст (HTF)**
- CHoCH на H1 при восходящем тренде на H4? Это сильный сигнал, но нужно смотреть на согласованность.
- Идеально, когда CHoCH происходит в сторону HTF тренда (смена на младшем ТФ совпадает со старшим).

**Шаг 3. Подтверждения**

| Подтверждение | Вес | Пояснение |
|---------------|-----|-----------|
| **Формирование нового свинга** | Высокий | После CHoCH появляется новый минимум выше (для бычьего) или максимум ниже (для медвежьего) |
| **Объём** | Средний | Объём на свече CHoCH выше среднего |
| **Kill Zone** | Средний | CHoCH в Kill Zone надёжнее |
| **Возврат в FVG/OB** | Очень высокий | Если после CHoCH цена заходит в FVG/OB – суперсигнал |

**Шаг 4. Точка входа**

*Пример для бычьего CHoCH (покупка):*
- CHoCH на баре i: close[i] > lastSwingHigh.
- **Вход:** на закрытии свечи i (рыночный) или на открытии следующей свечи.
- **Стоп-лосс:** на 3-5 пунктов ниже последнего Swing Low (перед разворотом) или ниже нового минимума.
- **Тейк-профит:** ближайший уровень ликвидности (BSL) или соотношение 1:3.

*Пример для медвежьего CHoCH (продажа):*
- CHoCH вниз: close[i] < lastSwingLow.
- Вход: продажа на закрытии.
- Стоп: выше последнего Swing High.
- Тейк: ближайший SSL.

**Шаг 5. Управление позицией**
- CHoCH часто является началом нового тренда, поэтому можно держать позицию долго.
- Трейлинг-стоп за новыми свингами.
- Частичная фиксация на 1:2.

### 7. Связь с другими инструментами из 48

| Инструмент | Связь с CHoCH |
|------------|---------------|
| **Swing High/Low** | CHoCH определяется через их пробой |
| **Liquidity Sweep** | CHoCH всегда предваряется Sweep противоположного свинга |
| **BOS** | BOS – продолжение, CHoCH – разворот |
| **FVG** | После CHoCH цена часто возвращается в FVG – точка входа |
| **Order Block** | CHoCH + OB = идеальный разворот |
| **Kill Zones** | CHoCH в Kill Zone надёжнее |
| **MSS (Market Structure Shift)** | Синоним CHoCH (у некоторых авторов) |

**Коэффициент надёжности:**
- CHoCH alone – 70-75%.
- CHoCH + возврат в FVG/OB – 85-90%.
- CHoCH + Kill Zone – 85-90%.
- CHoCH + объём + Kill Zone – 90-95%.

### 8. Психология и рыночная логика

**Почему CHoCH работает?**  

1. **Сбор ликвидности:** Перед разворотом крупные игроки сначала выносят стопы розничных трейдеров (Sweep). Это создаёт ликвидность для входа в противоположную сторону.
2. **Подтверждение контроля:** Закрытие за противоположным свингом показывает, что крупные игроки перехватили инициативу.
3. **Психологический барьер:** Розница, увидев пробой свинга, думает, что тренд продолжается, и открывает позиции. Когда цена разворачивается, они несут убытки, добавляя движение.

**Эмоции трейдера при CHoCH:**
- **Страх** – во время Sweep.
- **Неуверенность** – когда цена возвращается.
- **Желание войти** – на CHoCH.
- **Эйфория** – если угадал разворот.

**Правильная психология SMC:**  
Не пытаться предугадать CHoCH, а дождаться его формирования. Не входить на Sweep – входить только после закрытия за противоположным свингом.

### 9. Типичные ошибки и чек-лист

**Ошибки:**

1. **Принимать ложный CHoCH** – если цена закрылась за противоположным свингом, но следующий бар вернулся обратно.
2. **Вход на Sweep, а не на CHoCH** – слишком рано.
3. **Игнорирование HTF тренда** – CHoCH на M15 против H4 может быть лишь коррекцией.
4. **Не ждать нового свинга** – отсутствие подтверждения.
5. **Слишком близкий стоп** – цена часто делает ретест перед движением.

**Чек-лист для бычьего CHoCH:**

- [ ] Последний Swing Low и Swing High определены.
- [ ] Был Sweep последнего минимума (low < lastSwingLow).
- [ ] Цена закрылась выше последнего максимума (close > lastSwingHigh).
- [ ] Следующая свеча не закрылась ниже lastSwingHigh (или сформировала более высокий минимум).
- [ ] (Опционально) Есть FVG/OB в направлении входа.
- [ ] HTF тренд согласован (или CHoCH происходит на HTF).
- [ ] Kill Zone активна.
- [ ] R/R ≥ 1:2.
- [ ] Стоп за последним минимумом (или на 3-5 п. ниже low свечи Sweep).

---

### 10. Продвинутые нюансы

- **CHoCH на разных таймфреймах:** CHoCH на H4 значительно сильнее, чем на M15. Идеально, когда CHoCH на нескольких ТФ совпадают.
- **CHoCH и объём:** Истинный CHoCH часто сопровождается повышенным объёмом на свече закрытия. Низкий объём может указывать на ложный CHoCH.
- **CHoCH после новостей:** Если CHoCH произошёл сразу после важной новости, его надёжность ниже (спекулятивный всплеск).
- **CHoCH и PD Array:** CHoCH является сигналом добавить новую зону в персональный массив (зона разворота).
- **Автоматическое определение:** В SMC Pro CHoCH определяется автоматически. Можно настроить `InpSwingLookback` для чувствительности.

### 12. Резюме и шпаргалка

| Тип CHoCH | Условие | Вход | Стоп | Тейк |
|-----------|---------|------|------|------|
| Бычий | low < lastSwingLow и close > lastSwingHigh | Покупка на закрытии | Ниже lastSwingLow | BSL |
| Медвежий | high > lastSwingHigh и close < lastSwingLow | Продажа на закрытии | Выше lastSwingHigh | SSL |

**Мнемоническое правило:**  
*«Сначала вынос, потом возврат – CHoCH подаст разворота рать».*

### 14. Вопросы для самопроверки

1. **Что такое CHoCH?**  
   *Ответ:* Смена структуры рынка, подтверждённая закрытием за противоположным свингом после предварительного Sweep.
2. **Чем CHoCH отличается от BOS?**  
   *Ответ:* BOS – продолжение тренда, CHoCH – разворот.
3. **Какое обязательное условие предшествует CHoCH?**  
   *Ответ:* Sweep (пробой последнего свинга в противоположную сторону).
4. **Почему CHoCH считается надёжным сигналом?**  
   *Ответ:* Потому что он объединяет сбор ликвидности и смену структуры, что характерно для крупных игроков.
5. **Какой минимальный R/R?**  
   *Ответ:* 1:2.
