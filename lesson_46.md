### 1. Заголовок, номер и краткое определение

**Название:** Turtle Soup – ложный пробой с подтверждением  
**Номер:** Глава 46 (Часть 10. Дополнительные концепции, инструмент №2 из 4)  
**Суть одной фразой:** Turtle Soup – это специфический паттерн ложного пробоя (разворотный), при котором цена сначала пробивает последний свинг (Swing High или Swing Low), но следующая свеча закрывается в противоположную сторону, подтверждая разворот; назван в честь «черепаховой» торговой системы Ричарда Денниса, но адаптирован для SMC как сигнал раннего входа против пробоя.  
**Визуальное представление:** 🐢🍲 – стрелки (обычно зелёного цвета) на баре, где произошёл ложный пробой и подтверждение.

---

### 2. Исторический контекст

Название **Turtle Soup** отсылает к знаменитым «Черепахам» (Turtles) – группе трейдеров, обученных Ричардом Деннисом в 1980-х годах. Их система была трендовой, основанной на пробоях. Паттерн Turtle Soup предполагает, что многие трейдеры до сих пор используют пробои, и крупные игроки «варят черепаховый суп», то есть создают ложный пробой, чтобы заманить их в ловушку, а затем развернуть рынок.

В SMC Turtle Soup является частным случаем Swing Failure Pattern (SFP) и Inducement, но с дополнительным требованием подтверждения на следующей свече. В индикаторе SMC Pro реализован в модуле `AdditionalConcepts.mqh`.

---

### 3. Математическое определение и формальные условия

#### Определение Turtle Soup
Пусть `lastSwingHigh` и `lastSwingLow` – последние свинги.

**Бычий Turtle Soup (разворот вверх после ложного пробоя вниз):**
1. Цена пробивает последний минимум вниз: `low[i] < lastSwingLow`.
2. Затем цена закрывается **выше** этого минимума (возврат) – желательно на том же баре.
3. **Следующая свеча** (i+1) должна быть бычьей (close > open) и закрыться выше максимума свечи i (или выше `lastSwingLow`).
4. Вход на закрытии свечи i+1.

**Медвежий Turtle Soup (разворот вниз после ложного пробоя вверх):**
1. `high[i] > lastSwingHigh`.
2. Закрытие ниже `lastSwingHigh`.
3. Следующая свеча медвежья и закрывается ниже минимума свечи i.

#### Отличие от SFP
- SFP – это одна свеча с длинной тенью, закрытие внутри диапазона.
- Turtle Soup требует подтверждения на следующей свече (закрытие за экстремумом пробойной свечи).

#### Пример (бычий Turtle Soup):
- LastSwingLow = 1.1000.
- Свеча A: low=0.9995, close=1.1005 (выше lastSwingLow) – ложный пробой.
- Свеча B: бычья, close=1.1015 (выше high свечи A) – подтверждение.
- Вход: на закрытии свечи B.

### 4. Алгоритм обнаружения (реализация в SMC Pro)

В SMC Pro Turtle Soup реализован функцией `DetectTurtleSoup`.

**Псевдокод:**
```
Для каждого i:
    если IsSwingFailure(i, direction=1) и i+1 существует:
        если (направление вниз и close[i+1] < open[i+1] и close[i+1] < low[i]) -> медвежий Turtle Soup
        если (направление вверх и close[i+1] > open[i+1] и close[i+1] > high[i]) -> бычий Turtle Soup
```

**Реальный код из SMC Pro:**
```cpp
void DetectTurtleSoup(const datetime &time[], const double &open[], const double &high[],
                      const double &low[], const double &close[], int rates_total)
{
   for(int i=rates_total-3; i>=2; i--)
   {
      // медвежий Turtle Soup (ложный пробой вверх)
      if(IsSwingFailure(i, high, low, close, 1, 10) && i<rates_total-1)
      {
         if(close[i+1] < open[i+1] && close[i+1] < low[i])
         {
            // рисование стрелки вниз на баре i (или i+1)
            DrawArrow(g_prefix + "TURTLE_BEAR_" + IntegerToString(time[i]), time[i], high[i], -1, AC_COLOR_TURTLE, 2);
         }
      }
      // бычий Turtle Soup (ложный пробой вниз)
      if(IsSwingFailure(i, high, low, close, -1, 10) && i<rates_total-1)
      {
         if(close[i+1] > open[i+1] && close[i+1] > high[i])
         {
            DrawArrow(g_prefix + "TURTLE_BULL_" + IntegerToString(time[i]), time[i], low[i], 1, AC_COLOR_TURTLE, 2);
         }
      }
   }
}
```

**Цвет:** `AC_COLOR_TURTLE` по умолчанию `clrLime` (ярко-зелёный).

### 6. Торговая тактика (полный цикл сделки)

#### Стратегия «Turtle Soup»

**Шаг 1. Идентифицируем Turtle Soup**
- Находим бар ложного пробоя и следующий бар, который закрывается в противоположную сторону.

**Шаг 2. Вход**
- На закрытии подтверждающего бара (i+1).
- Стоп: за экстремумом ложного пробоя (выше максимума для медвежьего, ниже минимума для бычьего).
- Тейк: ближайший уровень ликвидности (противоположный свинг) или R:R 1:2.

**Пример (бычий Turtle Soup):**
- Бар A (ложный пробой): low=0.9995, close=1.1005.
- Бар B: close=1.1015 (подтверждение).
- Вход: покупка на 1.1015.
- Стоп: 0.9990 (ниже минимума A).
- Тейк: 1.1050 (ближайший BSL).

### 7. Связь с другими инструментами

| Инструмент | Связь |
|------------|-------|
| **Swing Failure Pattern** | SFP – основа Turtle Soup |
| **Inducement** | Turtle Soup – это Inducement с подтверждением |
| **CHoCH** | Часто после Turtle Soup следует CHoCH |
| **Liquidity Sweep** | Ложный пробой – это sweep |

**Коэффициент надёжности:** 75–85% при условии Kill Zone.

---

### 8. Психология

Turtle Soup требует дисциплины: не входить на первом ложном пробое, а ждать следующего бара. Это снижает количество ложных сигналов.

---

### 9. Типичные ошибки и чек-лист

**Ошибки:**
- Вход на баре ложного пробоя (без подтверждения).
- Игнорирование направления следующей свечи.
- Отсутствие Kill Zone.

**Чек-лист (бычий Turtle Soup):**
- [ ] Ложный пробой вниз (low < lastSwingLow, close > lastSwingLow).
- [ ] Следующая свеча бычья и закрывается выше high бара пробоя.
- [ ] Kill Zone активна.
- [ ] R/R ≥ 1:2.

### 10. Резюме и шпаргалка

| Тип | Условие | Вход | Стоп |
|-----|---------|------|------|
| Бычий | false break down + bullish follow‑up | На закрытии follow‑up | Ниже минимума ложного пробоя |
| Медвежий | false break up + bearish follow‑up | На закрытии follow‑up | Выше максимума ложного пробоя |

**Мнемоническое правило:**  
*«Черепаший суп – ложный пробой, а следом бар – вход готовь».*
