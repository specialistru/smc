## 📘 Глава 44. Trailing Stop – динамический стоп

> *Полная расширенная глава (объём ~15 000 знаков)*

---

### 1. Заголовок, номер и краткое определение

**Название:** Trailing Stop – динамический стоп  
**Номер:** Глава 44 (Часть 9. Модели входа и риск‑менеджмент, инструмент №3 из 3)  
**Суть одной фразой:** Trailing Stop (трейлинг-стоп) – это механизм автоматического перемещения стоп‑лосса вслед за ценой в направлении открытой позиции, позволяющий зафиксировать часть прибыли и при этом оставаться в сделке для получения большей прибыли; в SMC Pro используется простой трейлинг по последним свингам (Swing Low для лонга, Swing High для шорта).  
**Визуальное представление:** 📈🔄🛑 – линия стоп‑лосса, поднимающаяся (для лонга) или опускающаяся (для шорта) ступенчато, часто отображаемая пунктирной линией.

---

### 2. Исторический контекст

Трейлинг-стоп – стандартный инструмент риск-менеджмента, используемый во многих торговых платформах. В SMC его применение адаптировано под структуру рынка: стоп подтягивается не на фиксированное расстояние (например, 20 пунктов), а за последним значимым свингом (Swing Low для длинной позиции, Swing High для короткой). Это позволяет учитывать волатильность и не быть выбитым случайным шумом.

В индикаторе SMC Pro трейлинг-стоп реализован как опциональная функция (`InpShowTrailing`). Он не исполняет сделки автоматически, а лишь отображает на графике рекомендуемый уровень стоп‑лосса, который трейдер может переместить вручную.

---

### 3. Математическое определение и формальные условия

#### Виды трейлинга

**А. Фиксированный отступ (не используется в SMC Pro)**  
`SL_trail = current_price – offset` для лонга. Не учитывает структуру.

**Б. Трейлинг по свингам (используется в SMC Pro)**  
- Для длинной позиции: `SL = последний Swing Low` (или минимум, сформированный после входа).
- Для короткой позиции: `SL = последний Swing High`.

**В. Трейлинг по ATR**  
`SL = current_price – k * ATR` – более плавный, но не используется в SMC Pro.

#### Условия активации трейлинга
- Включается только когда позиция находится в прибыли (для лонга – текущая цена выше цены входа).
- Стоп перемещается только в сторону уменьшения убытка (никогда не расширяется).
- В SMC Pro трейлинг активируется после того, как цена прошла определённое расстояние (в коде – при `close[i] > entryPrice` для лонга).

#### Пример с реальными числами
- Лонг открыт по 1.1050. Последний Swing Low на момент входа = 1.1030 (стоп).
- Цена выросла до 1.1070, образовался новый Swing Low = 1.1055.
- Трейлинг-стоп подтягивается до 1.1055.

### 4. Алгоритм обнаружения (реализация в SMC Pro)

В `EntryModels.mqh` функция `UpdateTrailingStop`:

```cpp
void UpdateTrailingStop(const double &high[], const double &low[], const double &close[], int rates_total,
                        int type, double entryPrice, datetime entryTime)
{
   if(!InpShowTrailing) return;
   if(rates_total < 2) return;
   string tsName = g_prefix + "TRAILING_SL";
   double lastExtreme;
   if(type == 1)  // лонг
   {
      if(close[rates_total-1] > entryPrice) // цена выше входа (прибыль)
      {
         // ищем последний Swing Low для подтягивания
         for(int j = rates_total-2; j >= 0; j--)
            if(IsSwingLow(j, low, 5)) { lastExtreme = low[j]; break; }
         if(lastExtreme > 0)
            DrawHLineEntry(tsName, lastExtreme, ENTRY_COLOR_SL, STYLE_DOT, 1);
      }
   }
   else  // шорт
   {
      if(close[rates_total-1] < entryPrice)
      {
         for(int j = rates_total-2; j >= 0; j--)
            if(IsSwingHigh(j, high, 5)) { lastExtreme = high[j]; break; }
         if(lastExtreme > 0)
            DrawHLineEntry(tsName, lastExtreme, ENTRY_COLOR_SL, STYLE_DOT, 1);
      }
   }
}
```

**Пояснения:**  
- `InpShowTrailing` – входной параметр, включающий отображение.
- Используется фиксированный lookback=5 для поиска свингов (можно изменить).
- Линия перерисовывается каждый раз, когда появляется новый свинг.
- Старая линия не удаляется, а просто обновляется (функция `DrawHLineEntry` изменяет цену существующей линии).

---

### 5. Как это выглядит на графике

- Красная пунктирная линия (цвет `ENTRY_COLOR_SL`), которая перемещается вверх (для лонга) или вниз (для шорта) ступенчато.
- Надпись «Trailing SL» во всплывающей подсказке.
- Линия отображается только при включённом параметре.

### 6. Торговая тактика (использование трейлинга)

**Когда включать трейлинг:**
- После того как сделка вошла в прибыль на расстояние, равное первоначальному стопу (1:1).
- Либо с самого начала, но это может привести к преждевременному выходу.

**Правила использования:**
1. Открыть позицию по сигналу SMC (OB, FVG, CHoCH).
2. Установить начальный стоп (например, за зоной).
3. Как только цена проходит расстояние, равное риску (1:1), активировать трейлинг (вручную или автоматически).
4. Дождаться, когда цена сформирует новый свинг, и подтянуть стоп за ним.
5. Выход по трейлингу происходит, когда цена разворачивается и задевает уровень стопа.

**Пример (лонг EURUSD):**
- Вход 1.1050, начальный стоп 1.1030 (риск 20 п.).
- Цена достигает 1.1070 (прибыль 20 п.) → активируем трейлинг.
- Образуется новый Swing Low на 1.1055 → стоп подтягивается до 1.1055.
- Цена растёт до 1.1090, новый Swing Low 1.1075 → стоп 1.1075.
- Цена разворачивается и падает до 1.1075 → выход с прибылью 25 п. (вместо 20 п. при фиксации на 1:1).

### 8. Психология

Трейлинг-стоп помогает решить две эмоциональные проблемы:
- **Жадность** – позволяет оставаться в сделке, пока тренд силён.
- **Страх** – гарантирует, что часть прибыли не будет потеряна при развороте.

Но есть и ловушка: некоторые трейдеры начинают передвигать стоп вручную «чуть выше» нового свинга, нарушая правила. Лучше довериться автоматическому расчёту.

---

### 9. Типичные ошибки и чек-лист

**Ошибки:**
- Активация трейлинга слишком рано (ещё до 1:1) – частый вынос из-за шума.
- Использование слишком короткого lookback (1–2) – стоп будет слишком близко.
- Игнорирование волатильности – если свинги узкие, стоп может быть сбит.
- Не обновлять стоп при появлении нового свинга (лень) – потеря прибыли.

**Чек-лист для использования трейлинга:**
- [ ] Позиция уже в прибыли не менее 1:1.
- [ ] Определён последний свинг (Swing Low для лонга, Swing High для шорта).
- [ ] Установлен стоп за этим свингом (с небольшим запасом 3–5 п.).
- [ ] При появлении нового свинга стоп перемещается.
- [ ] Трейлинг отключён, если рынок вошёл в фазу консолидации (частые свинги).

---

### 10. Продвинутые нюансы

- **Lookback для свингов:** В SMC Pro используется 5. Для H1 это может быть мало; лучше 10–15. Можно модифицировать код.
- **Трейлинг по ATR:** Альтернативный метод – `SL = price - k*ATR`, где k=2-3. Более плавный, но не учитывает структуру.
- **Трейлинг по скользящей средней:** Не рекомендуется, так как запаздывает.
- **Порог активации:** Некоторые трейдеры активируют трейлинг только после достижения 1.5× начального риска.
- **Частичный трейлинг:** Закрыть 50% позиции на 1:1, а на остаток включить трейлинг.
