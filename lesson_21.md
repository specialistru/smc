## 📘 Глава 21. Vacuum Zone – узкая консолидация с последующим выбросом

---

### 1. Заголовок, номер и краткое определение

**Название:** Vacuum Zone – узкая консолидация с последующим выбросом  
**Номер:** Глава 21 (Часть 4. Ордерные блоки и зоны митигации, инструмент №8 из 9)  
**Суть одной фразой:** Vacuum Zone – это узкий ценовой диапазон, в котором цена консолидируется в течение нескольких свечей (обычно 5–15), после чего происходит резкий выброс (рывок) в одну из сторон, создавая эффект «вакуума», когда цена буквально всасывается из этого диапазона крупными игроками.  
**Визуальное представление:** 🌀📊 – прямоугольник, охватывающий узкий диапазон консолидации, часто жёлто-зелёного цвета, с подписью «Vacuum» и стрелкой направления выброса.

---

### 2. Исторический контекст и происхождение

Понятие **Vacuum Zone** не является классическим в SMC ICT; оно чаще встречается в авторских разработках сообщества SMC Pro и в некоторых школах анализа объёмов (Volume Spread Analysis, VSA). В VSA термин «vacuum» используется для описания ситуации, когда цена быстро проходит через зону без сопротивления, как через вакуум. В SMC Pro этот инструмент был формализован как зона консолидации, после которой следует **всасывающее движение** – агрессивный вход крупных игроков.

**Почему Vacuum Zone важна?**  
Консолидация (боковик) часто воспринимается розничными трейдерами как отсутствие направления. Но именно в таких узких диапазонах крупные игроки могут набирать позицию, не вызывая сильного движения. Когда они готовы, происходит «выброс» – цена резко покидает зону консолидации. Понимание Vacuum Zone позволяет:
1. Идентифицировать моменты накопления перед сильным движением.
2. Входить в сделку сразу после выброса (или на ретесте зоны).
3. Отличать истинные пробои от ложных.

---

### 3. Математическое определение и формальные условия

#### Определение Vacuum Zone
Рассмотрим последовательность баров от `i_start` до `i_end` (длина последовательности = N, обычно 5–20 баров).  
**Условия:**
1. **Узкий диапазон:** Разница между максимальным High и минимальным Low на этом участке мала относительно текущей волатильности:  
   `(maxHigh – minLow) / ATR(14) < 0.5` (или фиксированный порог, например, < 20 пунктов для EURUSD).
2. **Минимальная длина:** Не менее 5 баров.
3. **Отсутствие сильных трендовых свечей:** Все свечи в зоне имеют тело не более 30–50% от полного диапазона (то есть это «доджи» или свечи с маленьким телом).
4. **Выброс:** После зоны консолидации появляется свеча, диапазон которой **значительно** превышает средний диапазон зоны (например, в 2–3 раза) и которая закрывается за пределами зоны.
5. **Зона Vacuum** – прямоугольник от `minLow` до `maxHigh` на временном интервале от `time[i_start]` до `time[i_end]`.

#### Дополнительные условия:
- Чем длительнее консолидация, тем сильнее последующий выброс.
- Выброс может быть вверх или вниз. Направление определяется по закрытию выходящей свечи.
- После выброса часто происходит ретест зоны Vacuum (возврат к её границе).

#### Пример с реальными числами (EURUSD M15):
- 8 баров подряд: High колеблется между 1.1050 и 1.1060, Low между 1.1045 и 1.1055. Диапазон = 15 пунктов.
- ATR(14) = 25 пунктов → отношение 15/25 = 0.6 < 0.8 (можно считать узким).
- Следующий бар: High=1.1075, Low=1.1050, Close=1.1070 – выброс вверх.
- Vacuum Zone: прямоугольник 1.1045–1.1060 за последние 8 баров.

### 4. Алгоритм обнаружения (псевдокод + логика индикатора)

В SMC Pro отдельной функции для Vacuum Zone нет, но её можно реализовать. Приведём псевдокод и пример реализации.

**Псевдокод:**
```
Для каждого бара i от 5 до totalBars-1:
    Найти окно из N баров (N=10) заканчивающихся на i-1 (перед текущим баром)
    maxHigh = максимальный high в окне
    minLow = минимальный low в окне
    range = maxHigh - minLow
    avgRange = средний диапазон свечи в окне
    если range < 0.5 * ATR:
        // зона консолидации
        текущий бар i (выброс)
        если high[i] > maxHigh и close[i] > maxHigh:
            выброс вверх
            нарисовать Vacuum Zone (прямоугольник)
        если low[i] < minLow и close[i] < minLow:
            выброс вниз
```

**Пример реализации на MQL4 (для добавления в индикатор):**

```cpp
void DetectVacuumZone(const datetime &time[], const double &high[], const double &low[], const double &close[], int totalBars)
{
   if(!InpShowVacuum) return;
   int window = 10;  // длина окна консолидации
   double atr = iATR(Symbol(), PERIOD_CURRENT, 14, 0); // нужно вычислять заранее
   
   for(int i = window+1; i < totalBars; i++)
   {
      double maxHigh = high[i-window];
      double minLow = low[i-window];
      for(int j = i-window+1; j < i; j++)
      {
         if(high[j] > maxHigh) maxHigh = high[j];
         if(low[j] < minLow) minLow = low[j];
      }
      double zoneRange = maxHigh - minLow;
      if(zoneRange < atr * 0.5)  // узкий диапазон
      {
         // проверка выброса
         if(high[i] > maxHigh && close[i] > maxHigh) // выброс вверх
         {
            string name = g_prefix + "VACUUM_BULL_" + IntegerToString(time[i-window]);
            DrawRectangleZone(name, time[i-window], minLow, time[i-1], maxHigh, VACUUM_COLOR, 100);
            // можно нарисовать стрелку выброса
            DrawArrow(g_prefix+"VACUUM_ARROW_"+IntegerToString(time[i]), time[i], maxHigh, 1, clrWhite, 2);
         }
         if(low[i] < minLow && close[i] < minLow) // выброс вниз
         {
            string name = g_prefix + "VACUUM_BEAR_" + IntegerToString(time[i-window]);
            DrawRectangleZone(name, time[i-window], minLow, time[i-1], maxHigh, VACUUM_COLOR, 100);
            DrawArrow(g_prefix+"VACUUM_ARROW_"+IntegerToString(time[i]), time[i], minLow, -1, clrWhite, 2);
         }
      }
   }
}
```

### 6. Торговая тактика (полный цикл сделки)

Vacuum Zone используется для торговли **в направлении выброса**, часто на ретесте зоны или сразу после выброса.

#### Стратегия «Reteст Vacuum Zone»

**Шаг 1. Идентифицируем Vacuum Zone**
- Находим узкую консолидацию (5+ баров) и последовавший за ней выброс.

**Шаг 2. Определяем контекст (HTF)**
- Для выброса вверх: HTF тренд должен быть восходящим или хотя бы не нисходящим.
- Для выброса вниз: HTF нисходящий.

**Шаг 3. Ждём ретеста зоны Vacuum**
- После выброса цена часто возвращается к границе зоны (обычно к противоположной стороне от выброса).
- Например, при выбросе вверх цена может упасть обратно к верхней границе зоны.

**Шаг 4. Подтверждения**

| Подтверждение | Вес | Пояснение |
|---------------|-----|-----------|
| **Свечной паттерн** | Высокий | Пин-бар или поглощение на границе зоны |
| **CHoCH** | Высокий | После касания формируется новый экстремум в сторону выброса |
| **Объём** | Средний | На ретесте объём выше среднего |
| **Kill Zone** | Средний | Ретест в Kill Zone надёжнее |

**Шаг 5. Точка входа**

*Пример для выброса вверх (покупка на ретесте):*
- Зона Vacuum: 1.1050–1.1060.
- Выброс вверх до 1.1075, затем цена падает до 1.1060 (верхняя граница зоны).
- Свеча закрывается 1.1063 (пин-бар вверх).
- **Вход:** покупка на закрытии (1.1063).
- **Стоп-лосс:** на 3-5 пунктов ниже нижней границы зоны (1.1045).
- **Тейк-профит:** расстояние, равное высоте зоны (10 п.) × 2 = 1.1083, либо ближайший BSL.

*Пример для выброса вниз (продажа):*
- Зона 1.1045–1.1055, выброс до 1.1035, ретест до 1.1045, пин-бар вниз.
- Вход: продажа 1.1042.
- Стоп: 1.1060.
- Тейк: 1.1025.

**Шаг 6. Управление позицией**
- Vacuum Zone даёт надёжные сигналы, особенно при длительной консолидации.
- Частичная фиксация на 1:1, остальное до 1:2 или 1:3.

### 7. Связь с другими инструментами из 48

| Парный инструмент | Как усиливает Vacuum Zone | Пример |
|-------------------|---------------------------|--------|
| **Order Block** | Если зона Vacuum совпадает с OB | OB + консолидация = накопление |
| **FVG** | Выброс часто создаёт FVG | FVG после выброса |
| **Liquidity Sweep** | Выброс может быть sweep уровня | Sweep + Vacuum |
| **CHoCH** | Подтверждение после ретеста | Вход после CHoCH |
| **Kill Zones** | Vacuum в Kill Zone сильнее | Лондонская консолидация |

**Коэффициент надёжности:**
- Vacuum alone – 60-70%.
- Vacuum + свечной паттерн – 75-80%.
- Vacuum + CHoCH – 85-90%.
- Vacuum + OB + CHoCH – 90-95%.

### 8. Психология и рыночная логика

**Почему Vacuum Zone работает?**  

1. **Накопление энергии:** Узкая консолидация означает, что спрос и предложение временно уравновешены. Чем дольше длится баланс, тем сильнее будет дисбаланс при его нарушении.
2. **Манипуляция ликвидностью:** Крупные игроки собирают позицию внутри диапазона, не двигая цену. Выброс происходит, когда они готовы.
3. **Эффект «пружины»:** Цена как бы сжимается, а затем выстреливает.

**Эмоции трейдера:**
- **Скука** во время консолидации.
- **Страх** при выбросе (пропуск).
- **Надежда** на ретест.

**Правильная психология:**  
Использовать Vacuum Zone для входа на ретесте, а не в момент выброса.

### 9. Типичные ошибки и чек-лист

**Ошибки:**

1. **Вход сразу на выбросе** – часто ложный (быстро возвращается).
2. **Слишком широкая зона** – не Vacuum (более 0.8 ATR).
3. **Слишком короткая консолидация** (менее 5 баров) – слабый сигнал.
4. **Игнорирование CHoCH** – ретест может не привести к продолжению.
5. **Торговля против HTF тренда.**

**Чек-лист для выброса вверх:**

- [ ] Зона консолидации ≥5 баров, диапазон <0.5 ATR.
- [ ] Произошёл выброс вверх (high и close выше зоны).
- [ ] Цена вернулась к верхней границе зоны (или внутрь).
- [ ] Свечной паттерн на ретесте.
- [ ] CHoCH вверх.
- [ ] HTF восходящий.
- [ ] Kill Zone активна.
- [ ] R/R ≥ 1:2.

### 10. Продвинутые нюансы

- **Длина консолидации:** Оптимально 8–15 баров. Слишком длинная (>30) может означать, что рынок не определился.
- **Объём во время консолидации:** Должен быть низким или средним. Резкий рост объёма внутри зоны – признак скорого выброса.
- **Множественные Vacuum Zone:** Если после выброса цена снова входит в узкую консолидацию, это может быть повторное накопление.
- **Vacuum Zone на разных ТФ:** Vacuum на H1 сильнее, чем на M15.
- **Автоматическое определение:** В SMC Pro можно добавить параметр `InpShowVacuum` и функцию `DetectVacuumZone`.

### 14. Вопросы для самопроверки

1. **Что такое Vacuum Zone?**  
   *Ответ:* Узкая консолидация (5–20 баров) с последующим резким выбросом.
2. **Как измерить узость диапазона?**  
   *Ответ:* Отношение диапазона к ATR(14) < 0.5.
3. **Какой минимальный размер консолидации?**  
   *Ответ:* 5 баров.
4. **Когда лучше входить – на выбросе или ретесте?**  
   *Ответ:* На ретесте.
5. **Какой минимальный R/R?**  
   *Ответ:* 1:2.
