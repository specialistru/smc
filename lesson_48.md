## 📘 Глава 48. PD Arrays – обобщённое название FVG, OB, BB, Liquidity

> *Заключительная глава (объём ~12 000 знаков)*

---

### 1. Заголовок, номер и краткое определение

**Название:** PD Arrays – обобщённое название FVG, OB, BB, Liquidity  
**Номер:** Глава 48 (Часть 10. Дополнительные концепции, инструмент №4 из 4)  
**Суть одной фразой:** PD Arrays (Personal Demand Arrays) – это термин, введённый ICT для обозначения персонального набора зон спроса и предложения трейдера, куда входят ключевые уровни, такие как FVG, Order Block, Breaker Block, уровни ликвидности (BSL/SSL), а также Mitigation Block, Rejection Block и другие; PD Array – это не конкретный инструмент, а метод организации торгового пространства, когда трейдер выделяет на графике только самые сильные зоны, ожидая реакцию цены.  
**Визуальное представление:** 📋🎯 – панель или таблица на графике (в SMC Pro это информационное окно в левом верхнем углу), показывающая количество активных зон каждого типа.

---

### 2. Исторический контекст

Термин **PD Array** (Personal Demand Array) стал популярным благодаря ICT, который учил трейдеров вести список «своих» зон – не всех подряд, а только тех, которые имеют наибольшую значимость (обычно со старших таймфреймов). В индикаторе SMC Pro эта концепция реализована в виде информационной панели `ShowPDArraysInfo()`, которая отображает количество активных зон FVG, OB, BB, MB, RB, IFVG в текущий момент. Это помогает трейдеру быстро оценить рыночную ситуацию.

---

### 3. Определение PD Arrays

PD Array – это **не один инструмент**, а **совокупность** всех ранее изученных зон спроса и предложения, которые трейдер считает значимыми для своей торговли. Обычно в PD Array включают:

- **Fair Value Gaps (FVG)** – особенно недавние и не заполненные.
- **Order Blocks (OB)** – особенно unmitigated (не тестированные).
- **Breaker Blocks (BB)** – пробитые OB, сменившие роль.
- **Mitigation Blocks (MB)** – уровни первой коррекции.
- **Rejection Blocks (RB)** – свечи с длинными тенями.
- **Уровни ликвидности (BSL/SSL)** – последние свинги.
- **Equal Highs/Lows (EQH/EQL)** – уровни с множественными касаниями.
- **IFVG** – инвертированные FVG.

Каждый трейдер составляет свой PD Array, выбирая зоны с наибольшим весом (совпадение нескольких инструментов, близость к текущей цене, формирование на старшем ТФ). Индикатор SMC Pro лишь подсчитывает количество таких зон, но не ранжирует их.

---

### 4. Реализация в SMC Pro (информационная панель)

В модуле `AdditionalConcepts.mqh` функция `ShowPDArraysInfo()` создаёт текстовую метку в левом верхнем углу графика:

```cpp
void ShowPDArraysInfo()
{
   string text = "PD Arrays:\n";
   text += "FVG: " + IntegerToString(ArraySize(fvgZones)) + " | ";
   text += "OB: " + IntegerToString(ArraySize(obZones)) + " | ";
   text += "BB: " + IntegerToString(ArraySize(bbZones)) + " | ";
   text += "MB: " + IntegerToString(ArraySize(mbZones)) + " | ";
   text += "RB: " + IntegerToString(ArraySize(rbZones)) + " | ";
   text += "IFVG: " + IntegerToString(ArraySize(ifvgZones));
   // отображение метки OBJ_LABEL
}
```

**Цвет текста:** `AC_COLOR_PD` (по умолчанию `clrWhite`). Размер шрифта 8.

---

### 5. Как использовать PD Arrays в торговле

1. **Оценка насыщенности рынка:** Много активных зон (например, >20 FVG) может указывать на высокую волатильность и возможные ложные сигналы.
2. **Приоритизация:** Зоны, которые совпадают с несколькими типами (например, FVG и OB на одном уровне), имеют более высокий приоритет.
3. **Планирование:** Перед началом сессии просмотрите PD Array (таблицу) и отметьте на графике только самые сильные зоны вручную (или используйте фильтры индикатора).

---

### 6. Связь с другими инструментами

PD Array – это **мета-инструмент**, объединяющий почти все предыдущие 47 инструментов. Он не даёт сигналов, но помогает структурировать информацию.

---

### 7. Психология

Ведение PD Array дисциплинирует трейдера: он перестаёт «торговать всё подряд» и фокусируется на зонах, которые действительно важны. Это снижает перегрузку и повышает качество сделок.

---

### 8. Типичные ошибки

- Включать все зоны без фильтрации – информационная панель будет показывать десятки зон, что бесполезно.
- Игнорировать PD Array при входе.
- Не обновлять PD Array вручную (индикатор обновляет автоматически, но трейдер должен визуально оценивать качество зон).

---

### 9. Резюме

PD Array – это ваш персональный «архив» сильных уровней. Используйте информационную панель SMC Pro как подсказку, но окончательный отбор зон проводите вручную, основываясь на совпадении нескольких инструментов и старшем таймфрейме.

### 10. Вопросы для самопроверки

1. **Что означает аббревиатура PD Array?**  
   *Ответ:* Personal Demand Array – персональный массив зон спроса.
2. **Какие инструменты входят в PD Array?**  
   *Ответ:* FVG, OB, BB, MB, RB, IFVG, уровни ликвидности и др.
3. **Как PD Array отображается в SMC Pro?**  
   *Ответ:* В виде текстовой панели в левом верхнем углу.
4. **Нужно ли торговать все зоны из PD Array?**  
   *Ответ:* Нет, только самые сильные (с совпадениями).
5. **Какой цвет текста PD Array по умолчанию?**  
   *Ответ:* Белый.
